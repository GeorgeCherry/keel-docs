# Keel v0.368.0

## Release Date

29 September, 2023

## Summary

We are excited to announce the release of Keel `v0.368.0`. This update ships a new feature to Keel.

## Events

With events you are able to asynchronously execute a subscriber function at the point when an _event_ happens in your application. 

Read all about [Eooks](/events) here.
